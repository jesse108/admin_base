版本：0.4.1

发布时间：2015-01-28

基本介绍：

这是SLS SDK for PHP 的第一个对外发布版本。SLS SDK for PHP是阿里云简单日志服务
（Simple Log Service）API的PHP编程接口，提供了对于SLS Rest API所有接口的封装
和支持，帮助PHP开发人员更快编程使用阿里云SLS服务。

具体功能：

1. 通过Request-Response风格的接口封装SLS Rest API。
2. 实现SLS API请求的数字签名
3. 实现SLS API的Protocol Buffer格式发送日志
4. 支持SLS API定义的数据压缩方式
5. 使用PHP异常统一处理错误

环境要求：
1. PHP 5.2.1及其以后版本

支持API版本：

1. SLS API 0.4.0
